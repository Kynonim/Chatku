# Chatku

A new Flutter project.

# Install
* 1. Download git, flutter, flutter sdk, and etc
* 2. Clone this project
```bash
git clone https://github.com/Kynonim/chatku.git
cd chatku
```

# Run the app
```bash
flutter pub get
flutter run
```
